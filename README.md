# Smart Flight Delay Prediction System

##  Overview
This project is a machine learning-based web application that predicts the likelihood of a flight delay based on airline, departure time, and weather conditions. The system is deployed using Streamlit for interactive user input and real-time predictions.



## Problem Statement
Flight delays cause operational inefficiencies and passenger inconvenience. The goal of this project is to build a predictive system that estimates delay risk using historical flight-related factors.



## Solution Approach
A supervised machine learning model (Random Forest Classifier) is trained on structured flight data to classify whether a flight is likely to be delayed or not.



## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Random Forest Classifier



## Features Used
- Airline
- Departure Hour
- Weather Conditions


## Workflow
1. Data creation and preprocessing  
2. Label encoding of categorical variables  
3. Model training using Random Forest  
4. Model evaluation  
5. Streamlit UI deployment  
6. Real-time prediction interface  



##  Output
The system outputs:
- "High Chance of Delay"
- "Low Chance of Delay"

based on user inputs.
