# Smart Flight Delay Prediction System

## Overview
This project predicts whether a flight will be delayed based on airline, departure time, and weather conditions using Machine Learning.

## Objective
To build a simple aviation analytics model that demonstrates predictive capability using structured flight data.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Random Forest Classifier

## Workflow
1. Data creation and preprocessing
2. Label encoding of categorical features
3. Model training using Random Forest
4. Model evaluation using classification metrics
5. Prediction on sample inputs

## Output
- Delay prediction (Yes/No)
- Model performance report

## Future Improvements
- Use real airline datasets
- Add weather API integration
- Build web app using Streamlit
- Deploy model online
